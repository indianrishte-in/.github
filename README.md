# Indian Rishte

**A dignified matrimonial platform for meaningful connections within the Muslim community.**

We build tools that help families find compatible life partners — with trust, verified identities, and privacy at the core.

🌐 **[indianrishte.in](https://indianrishte.in)**

---

## What We're Building

A full-stack matrimonial profile discovery platform serving the Muslim community.

- **Browse & Search** — Filter by gender, age group, marital status, city, and district
- **Verified Profiles** — Identity verification for trust and safety
- **User Dashboard** — Create and manage your own profile with a guided multi-step form
- **Privacy First** — Controlled data visibility and secure authentication via Clerk
- **Performance** — 4-layer caching (SWR → CDN → Redis → PostgreSQL) for fast, reliable browsing
